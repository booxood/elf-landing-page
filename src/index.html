ELF
HTML5 多场景应用模板
ELF 是凹凸实验室出品的一套面向开发者的灵活可扩展的 HTML5 多场景应用模板，
可用来制作各种 HTML5 场景营销活动页面；
它基于 Webpack 进行自动化构建，目前共有4个组件以及4各官方案例模板；
利用 ELF 提供的命令行工具 elf-cli，开发者可自由通过模板和组件的组合来快速定制开发 HTML5 场景应用。
# Step 1. 安装 elf-cli
$ npm install -g elf-cli
# 依赖的包比较多，安装非常耗时，推荐下面这行命令进行安装
# mac/linux
$ SASS_BINARY_SITE=https://npm.taobao.org/mirrors/node-sass/ PHANTOMJS_CDNURL=https://npm.taobao.org/mirrors/phantomjs/ npm install -g elf-cli --registry=https://registry.npm.taobao.org
# windows
$ npm install -g elf-cli --registry=https://registry.npm.taobao.org --SASS_BINARY_SITE=https://npm.taobao.org/mirrors/node-sass/ --PHANTOMJS_CDNURL=https://npm.taobao.org/mirrors/phantomjs/
# Step 2.创建项目目录
$ mkdir demo && cd demo
# Step 3.执行「elf init -t 案例名」基于指定模板初始化项目，可执行「elf list」查看所有模板，默认 base
$ elf init -t swiper
# Step 4. 安装依赖并执行
$ npm i && npm start
特性
无需构建配置文件并可进行开发
  • 微信友好性
    针对微信做了友好性兼容，支持快速配置分享至微信的封面、标题及描述
  • 响应式
    基于 REM 或 ZOOM 的响应式设计，支持 PX 自动转换 REM
  • 样式预处理
    支持 Sass, Less 或 Stylus 样式预处理语言及 Autoprefixer
  • Webpack 构建
    基于 Webpack 进行自动化构建，支持雪碧图合并、资源压缩、Hot-Reload 等特性
